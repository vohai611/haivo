Vietnam statistic dashboard with R

August 6, 2021 · 1 min · Hai Vo
Table of Contents

In this project, I built an small Shiny (an R framework to build webapp) dashboard to visualize general statistic data of Vietnam. The source of data came from Vietnam general statistics office.

The dashboard now hosted on shinyapps.is but it can be deploy on any others platform. You could browser my dashboard source code at this github repo. The repository also had a Docker file to build the dashboard imagine.
